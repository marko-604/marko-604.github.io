Emotion Recognition & LLM Integration

Integration Process

The system uses a deep learning model (ResNet-50) for emotion classification and a local LLM to interpret results in real-time.

Why Use an LLM?

The LLM provides context-aware insights by analyzing user intent and emotional trends, enabling more meaningful feedback.

Future Applications

Plans include expanding the model for gameplay behavior analysis, mental health monitoring, and interactive AI-driven experiences.
